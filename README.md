# Pharaoh Pose — Prototype Phase 0

Un danseur égyptien court dans une pyramide ; des murs percés de silhouettes foncent sur lui.
Prends la bonne pose au bon moment — et la survie devient une danse.

**Jouer :** https://thevisitappsolution-create.github.io/pharaoh-pose/

- Mur fin → tape la pose (boutons 1-2-3-4 ou tactile)
- Mur épais → reste appuyé jusqu'au bout
- Pose prise au dernier moment → **Perfect**, points doublés

Prototype hyper-casual (Phase 0 « fun-first ») — un seul fichier `index.html`, zéro dépendance.
